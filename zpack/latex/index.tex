ZPack is a simple, general-\/purpose archive file format with lossless data compression (similar to zip, rar, etc.). It is designed with performance in mind and uses fast compression algorithms such as \href{https://github.com/facebook/zstd}{\texttt{ zstd}} and \href{https://github.com/lz4/lz4}{\texttt{ lz4}}.\hypertarget{index_autotoc_md0}{}\doxysection{Documentation}\label{index_autotoc_md0}
The specifications for the ZPack file format can be found here

The API documentation can be generated using Doxygen. An online copy of it can be found \href{https://leadrdrk.eu.org/zpack}{\texttt{ here}}\hypertarget{index_autotoc_md1}{}\doxysection{Installing}\label{index_autotoc_md1}
Prebuilt Windows and Linux binaries are available on the \href{https://github.com/LeadRDRK/ZPack/releases}{\texttt{ Releases}} page.\hypertarget{index_autotoc_md2}{}\doxysection{Compiling}\label{index_autotoc_md2}
The CMake lists in the root directory can be used to build both the library and the command line utility.

A few options are available when configuring with CMake. Here are some common options\+:
\begin{DoxyItemize}
\item {\ttfamily ZPACK\+\_\+\+BUILD\+\_\+\+PROGRAMS}\+: Build the ZPack command line utility. Default\+: {\ttfamily ON}
\item {\ttfamily ZPACK\+\_\+\+BUILD\+\_\+\+TESTS}\+: Build the ZPack unit tests. Default\+: {\ttfamily OFF}
\item {\ttfamily ZPACK\+\_\+\+DISABLE\+\_\+\+ZSTD}\+: Disable zstd support. Default\+: {\ttfamily OFF}
\item {\ttfamily ZPACK\+\_\+\+DISABLE\+\_\+\+LZ4}\+: Disable LZ4 support. Default\+: {\ttfamily OFF}
\end{DoxyItemize}

More options can be found in \href{CMakeLists.txt}{\texttt{ CMake\+Lists.\+txt}}

To use the library in your CMake project\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{set(ZPACK\_BUILD\_PROGRAMS OFF) \# optional}
\DoxyCodeLine{add\_subdirectory(path/to/zpack)}
\DoxyCodeLine{}
\DoxyCodeLine{target\_include\_directories(target PRIVATE path/to/zpack/lib)}
\DoxyCodeLine{target\_link\_libraries(target PRIVATE zpack)}

\end{DoxyCode}
