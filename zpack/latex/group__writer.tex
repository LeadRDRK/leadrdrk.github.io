\doxysubsection{Detailed Description}
The archive writer.~\newline
Thread safety\+: {\bfseries{Not thread safe.}}~\newline
zpack\+\_\+writer depends on a write\+\_\+offset variable to know where to write next. Running multiple write operations at the same time will result in undefined behavior.~\newline
Note that there are no restrictions on the ordering of each block while writing the archive, so make sure to follow the correct order.~\newline
(header -\/\texorpdfstring{$>$}{>} data header -\/\texorpdfstring{$>$}{>} files -\/\texorpdfstring{$>$}{>} cdr -\/\texorpdfstring{$>$}{>} eocdr)
